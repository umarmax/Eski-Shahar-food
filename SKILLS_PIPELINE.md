## 🔗 AUTONOMOUS SKILL CHAINS

### Pipeline Rules:
1. **Never use a single skill alone** — always chain 2-4 skills per task
2. **Auto-detect context** from user input and select the right skill chain
3. **Always finish with quality pass** (copy-editing for text, caveman-review for code)
4. **Prioritize revenue-generating skills** or immediate problem solvers
5. **End every task with C-Level Advisory opinion**
